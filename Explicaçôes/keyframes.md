## Keyframes 

<p> Keyframes e uma função do css , que usa as propriedades , defindo o tempo de execusão para  animação.</p>


```css
 
@keyframes mudarCor {

 0% {

     background-color: seagreen;

 }

 100%{
    
     background-color: green ;

 }

}

```

<p> Para declarar no css. </p>

```css 

body{

    background-color : green;
    animation: mudarCor 2s linear ;

 }

```
